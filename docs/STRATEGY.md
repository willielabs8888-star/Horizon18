
# Horizon18 — Strategic Direction

Horizon18 is not a calculator. It's a decision intelligence platform for the biggest financial decision most teenagers will face.

## Core Moat
- Clear, research-backed modeling (BLS, College Board, NACE, DFAS data)
- Clean UX with interactive charts, hover tooltips, and multi-instance comparison
- Trust-based brand — neutral, educational, no upselling
- Institutional relationships (long-term)

## Monetization (Later)
- Parent access tier
- School licensing
- Advanced modeling features (inflation, school-level lookup, Monte Carlo)

## Current Status
- **Web app COMPLETE**: Full React frontend + Python backend, interactive charts, multi-instance support
- **96 engine tests + 9 API tests** passing
- **Next phase**: Deploy to production, validate with schools and counselors, build partnerships

## Phase 4 Priority
1. Production infrastructure (hosting, monitoring, logging)
2. School district partnerships and pilot programs
3. Institutional feedback loop for model refinement
